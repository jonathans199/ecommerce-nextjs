To start MongoDB Server:

```bash
# to create and start Mongo container
$ docker-compose up

# to stop mongo docker container
$ docker-compose down

# to seed MongoDb with Fake data
$ yarn seed
# or
$ npm run seed

```

## Tech Stack

**Frontend:** NextJS, TailwindCSS

**Database:** MongoDB - Mongoose
